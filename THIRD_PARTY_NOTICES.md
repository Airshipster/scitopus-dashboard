## Visualization Pattern Attribution

The Race mode uses the general bar chart race visualization pattern, a well-known animated ranking format popularized in the D3/Observable ecosystem.

No direct source link to a specific original GitHub bar chart race repository was found in the current project files. For that reason, this repository is presented as a standalone custom implementation rather than a fork. If an exact upstream source is later identified, add it here with its author, URL, license, and the scope of reuse.
